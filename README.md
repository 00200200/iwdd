# iwdd

## Requirements

- **Python 3.11+**
- **uv package manager** - https://docs.astral.sh/uv/getting-started/installation/


## Quick Start

1. **Setup:**

   ```bash
   # Clone the repository
   git clone https://github.com/00200200/iwdd.git
   cd iwdd

   # Install dependencies
   uv sync
   ```
